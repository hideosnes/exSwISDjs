## Under Development

This is a [Node.js](https://nodejs.org/en/) module and will be available through the
[npm registry](https://www.npmjs.com/).

Before installing, [download and install Node.js](https://nodejs.org/en/download/).
Node.js 22 or higher is required.

```bash
// npm install exSwISD --safe // under_development
```

## Features

  * Robust routing
  * Efficient P2P routing
  * Focus on high performance
  * HTTP helpers (redirection, caching, etc)
  * Content negotiation

## Progenitor

Based on [`express.js`](https://github.com/expressjs)
